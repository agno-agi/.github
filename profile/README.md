## Start here

- [**agno**](https://github.com/agno-agi/agno). The SDK for building agent platforms. Build agents using any framework, run them as production services, manage everything from a single control plane. Own your agent stack: your cloud, your data, your database.
- [**docs**](https://docs.agno.com). Documentation for everything Agno. Build your [first agent in 20 lines of code](https://docs.agno.com/first-agent).
- [**AgentOS UI**](https://os.agno.com). The control plane for your agent platform. Chat with agents, inspect sessions, traces, memory, and evals.

## Application templates. Products you can deploy today

Full agent products built on AgentOS. Fork one, make it yours.

| App | What it does |
|---|---|
| [dash](https://github.com/agno-agi/dash) | A self-learning data agent — grounds answers in 6 layers of context and improves with every query. |
| [coda](https://github.com/agno-agi/coda) | A code companion that lives in Slack — answers architecture questions, reviews PRs, triages issues. |
| [scout](https://github.com/agno-agi/scout) | An open-source company brain — navigates Slack, Drive, and wikis, and builds its own wiki and CRM as it learns. |
| [context](https://github.com/agno-agi/context) | A self-hosted context manager — one source of work context for you and your AI tools. |
